# HTML to BB Converter for VS Code

Конвертирует выделенный HTML в BB-код для форумов MyBB/RusFF.

## Возможности

- Преобразует `<div class="name">` в `[block=name]`
- Преобразует `<h1>` в `[block=bh1]`
- Преобразует `<span>` в `[block=bspan]`
- Преобразует `<a href="...">` в `[url=...]`
- Преобразует `<img>` в `[img][/img]`
- Сохраняет структуру вложенных блоков

## Установка

1. Скачайте `.vsix` файл из релизов
2. В VS Code: Extensions → ... → Install from VSIX
3. Выберите скачанный файл

## Использование

1. Выделите HTML код
2. Нажмите правую кнопку мыши
3. Выберите "BB Converter: Конвертировать выделенный HTML"
